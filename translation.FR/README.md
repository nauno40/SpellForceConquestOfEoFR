# Traduction Française pour SpellForce: Conquest of Eo

Ce mod fournit une traduction française complète et corrigée pour **SpellForce: Conquest of Eo**.

## Description

Ce projet vise à offrir une localisation française intégrale et soignée pour le jeu. Tous les textes, y compris les dialogues, les descriptions de sorts, d'objets, de quêtes et l'interface utilisateur, ont été traduits. Une attention particulière a été portée à la cohérence du lore et à la correction des bugs de traduction présents dans les versions précédentes.

## Contenu et Extensions

Cette traduction a été réalisée avec l'assistance de l'IA **Gemini 3** (Google) pour garantir une haute qualité contextuelle.

Elle couvre le jeu de base ainsi que toutes les extensions sorties à ce jour :
- **Demon Scourge**
- **Weaver's Realms**
- **Children of Nor**

## Installation

1. Assurez-vous que le jeu **SpellForce: Conquest of Eo** est installé.
2. Localisez votre dossier de documents du jeu. Il se trouve généralement ici :
   `C:\Users\[VotreNom]\Documents\My Games\Spellforce - Conquest of Eo\content\`
   *(Si le dossier `content` n'existe pas, créez-le).*
3. Copiez le dossier `translation.FR` de ce mod à l'intérieur du dossier `content`.
   - Le chemin final devrait être : `...\Spellforce - Conquest of Eo\content\translation.FR\`
4. Lancez le jeu. Si la traduction n'est pas détectée automatiquement, vérifiez les options de langue dans le menu principal.
